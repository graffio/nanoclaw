**Replies:**

> **Stephen Tobin** (06:54): The account return includes cash. As an example lets suppose I make $1,000 in trading profit if I own $10,000 in stocks then the SW return would be 10%. However my account might have $10,000 in stocks and $5,000 in cash for a total of $15,000 so the account return is 6.6%. (in both cases it is the profit divided by the starting figure, so 1,000 / 10,000 = 0.1 and 1,000 / 15,000 = 0.66)
