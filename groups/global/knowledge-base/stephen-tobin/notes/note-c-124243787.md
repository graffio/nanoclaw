# Note -- June 9, 2025

Quantum security, a key use case for photon quantum computing looks less secure than expected. The first real life test performed by the Chinese used the BB84 Quantum Key Distribution system thought to be mathematically secure in a communication satellite. However the pulse lasers used to transmit the data has shown a vulnerability that would allow a hacker with the right equipment to crack the message 98% of the time. The mathematics seems perfect but the engineering cannot deliver the required precision, a 100 pico second delay allowed the well known Photon Number Splitting attack to succeed. Quantum is becoming a Wild West for investors, companies are making claims that fail almost without exception.
